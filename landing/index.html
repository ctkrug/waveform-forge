Client-side audio, start to finish

See your audio. Trim it. Ship it. Never leave the tab.

Drop in an MP3, WAV, AAC, FLAC, OGG, or M4A file and Waveform Forge decodes it, draws a waveform and spectrogram, lets you trim to the sample, and exports the result — all without a single byte leaving your machine.

Panel spec

DECODE
Native Web Audio decode for MP3 / WAV / AAC / OGG, with an ffmpeg.wasm fallback for anything the browser won’t touch — FLAC, M4A, and other exotic containers.
VISUALIZE
A hand-written FFT drives a synced waveform + spectrogram pair, both zoomable and pannable, with labeled time and frequency axes.
TRIM
Drag, keyboard-nudge, or touch the in/out handles for a sample-accurate selection, with instant preview playback of just that region.
EXPORT
Transcode the trimmed selection to MP3, AAC, or WAV via ffmpeg.wasm and download it straight to disk — no server round-trip, ever.

Your file never leaves your machine

Every other "convert my audio" tool on the web makes you upload the file, wait in a queue, and hope the server is up. Waveform Forge decodes, analyzes, trims, and re-encodes entirely in the browser tab — the only network request is the one-time download of the ffmpeg.wasm engine, cached after first use.
